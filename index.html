System Routes Overview

Important Note

The terms "Manager" and "QC" refer to the same role in the system. Any tasks related to Quality Control (QC) are performed by users with the Manager role. Throughout the system, assume Manager = QC.

Operator Portal

The main page for production operators to create and manage batch records before QC/Manager approval.

Features:

  • Create new batch production records
  • Record components, packaging, and ingredients
  • Document start and end times
  • Submit records for Manager (QC) verification

Staging Area (Manager/QC)

A central location for Managers (QC) to review, approve, and manage all batch records.

Features:

  • View and search all batch records
  • Perform QC reviews (Manager acts as QC)
  • Verify components, packaging, and documentation
  • Approve or reject batches based on quality standards
  • Add QA notes and ensure compliance
  • Monitor and track approval status

üìç Routes:
- Main listing: /staging
- Detailed review & approval: /staging/{record_id}

Administration

System administration and user management functions.

Features:

  • Create new user accounts
  • Manage user roles (operator, manager/QC, admin)
  • Handle system maintenance tasks
  • Manage and maintain data integrity
